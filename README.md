# Riya AI Assistant 🤖

A powerful personal AI assistant built from scratch using Python and Neural Networks.

## Features
- 🧠 Neural Network from scratch
- 🤖 TinyLLaMA AI brain
- 🌤️ Live weather
- 🌐 Web search
- 💻 Code assistant
- 📄 File reader
- 🧠 Long term memory
- 😊 Emotion detection
- 📷 Face recognition
- 🎨 Image generation
- 🔊 Voice input and output
- 🌍 Multi language

## Run Riya
pip install -r requirements.txt
python train.py
python riya_final.py
